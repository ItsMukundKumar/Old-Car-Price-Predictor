# 🚗 Old Car Price Prediction App

A **Machine Learning web application** built with **Python and Streamlit** that predicts the selling price of a used car based on various attributes such as car name, year, fuel type, transmission, and kilometers driven.

The model is trained using a **Decision Tree Regressor** and deployed with an interactive **Streamlit interface**.

---

## 📊 Features

- Predict selling price of used cars
- Interactive web interface using Streamlit
- Decision Tree Machine Learning model
- Label Encoding for categorical features
- Real-time prediction

---

## 🧠 Machine Learning Model

Model used:

- **Decision Tree Regressor**

Input features:

- Car_Name
- Year
- Present_Price
- Kms_Driven
- Fuel_Type
- Seller_Type
- Transmission

Target variable:

- **Selling_Price**
