<h1 align="center">🚗 Old Car Price Prediction App</h1>

<p align="center">
A <b>Machine Learning web application</b> built using <b>Python</b> and <b>Streamlit</b> that predicts the 
<b>selling price of a used car</b> based on key vehicle attributes.
</p>

<p align="center">
The application uses a <b>Decision Tree Regression model</b> trained on historical car data and provides 
real-time predictions through an interactive web interface.
</p>

<hr>

<h2>📊 Key Features</h2>

<ul>
<li>Predict resale price of used cars</li>
<li>Interactive web interface built with Streamlit</li>
<li>Machine learning model using <b>Decision Tree Regressor</b></li>
<li>Label Encoding for categorical features</li>
<li>Instant real-time predictions</li>
</ul>

<hr>

<h2>🧠 Machine Learning Model</h2>

<h3>Algorithm</h3>
<ul>
<li><b>Decision Tree Regressor</b></li>
</ul>

<h3>Input Features</h3>

<table>
<tr>
<th>Feature</th>
<th>Description</th>
</tr>

<tr>
<td>Car_Name</td>
<td>Name or model of the car</td>
</tr>

<tr>
<td>Year</td>
<td>Manufacturing year of the vehicle</td>
</tr>

<tr>
<td>Present_Price</td>
<td>Current ex-showroom price (in lakhs)</td>
</tr>

<tr>
<td>Kms_Driven</td>
<td>Total kilometers driven</td>
</tr>

<tr>
<td>Fuel_Type</td>
<td>Petrol / Diesel / CNG</td>
</tr>

<tr>
<td>Seller_Type</td>
<td>Dealer or Individual</td>
</tr>

<tr>
<td>Transmission</td>
<td>Manual or Automatic</td>
</tr>

</table>

<h3>Target Variable</h3>

<ul>
<li><b>Selling_Price</b> — Predicted resale value of the car</li>
</ul>

<h2>⚙️ Installation</h2>

<h3>1. Clone the Repository</h3>

<pre>
git clone https://github.com/ItsMukundKumar/Old-Car-Price-Predictor.git
cd Old-Car-Price-Predictor
</pre>

<h3>2. Install Dependencies</h3>

<pre>
pip install -r requirements.txt
</pre>

<h3>3. Run the Application</h3>

<pre>
streamlit run app.py
</pre>

<p>The application will start locally and open in your browser.</p>
